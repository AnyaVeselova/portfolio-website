HTML & CSS basics

one-page website

By doing this project I've learned the fundamentals of HTML and CSS. Within HTML course I've learned how to place a content on a page and also how to organize the content(semantic HTML). The topics covered are - document structure, inline and block level elements, nesting, attributes and links, images, lists to name a few.

As for CSS, I've acquired the understanding of its basic syntax, inheritance, cascading behaviour, the box model, selectors(ID, class), spans, divs, centering content, got to know a little bit of flexbox, background images and relative file paths.
